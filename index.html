Agenda de Oficina

Ingresá la URL del script de Google Apps para conectar la agenda compartida.

📋 Cómo obtener la URL:

  1. Abrí tu Google Sheet de la agenda
  2. Ir a Extensiones → Apps Script
  3. Pegá el código del archivo Code.gs
  4. Click en Implementar → Nueva implementación
  5. Tipo: Aplicación web · Ejecutar como: Usuario que accede
  6. Copiá la URL generada y pegala acá abajo

Esta URL solo se guarda en tu dispositivo. Cada integrante del equipo debe configurarla la primera vez.
